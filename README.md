# X Monetization Bot - Desktop App

Simple Tkinter desktop application for Twitter automation.

## Installation

1. Install Python 3.8+
2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the app:
```bash
python main.py
```

## Features

- Tkinter GUI
- SQLite database
- AI-powered replies using Groq API
- Template-based replies
- Works with Twitter lists
- Target: 1000 replies per day
